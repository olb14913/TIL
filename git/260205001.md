- commit 전에 체크해야 할 것들을 자동 수행하도록 도와주는 도구

## 2. branch

### 2.1. branch 확인
```shell
git branch```

### 2.2. branch 생성
```shell

```

### 2.3. branch 전환
```shell

```

### 2.4. branch 삭제 
```shell
git branch -D {branch_name}
```

- merge > vi를 통해 충돌파일(ex : <<<<<<,======) 편집 및 저장 > git add {file_name} > git commit
